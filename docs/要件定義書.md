**【システム要件定義書（第1版）】**

**1. システム概要**
マークダウンでの執筆機能を備えた、個人用のブログおよびコンテンツ管理システム（CMS）です。Next.jsのApp Routerによる高速なレンダリングと、Supabaseを利用したセキュアなバックエンドを構築します。

**2. 機能要件**

* **公開画面 (Public)**
* 記事一覧表示（すべての記事、およびタグ・カテゴリ別の絞り込み表示）
* 記事詳細表示（Markdown本文のHTMLレンダリング）


* **管理画面 (Admin)**
* Supabase Authを用いたセキュアな管理者ログイン機能
* 記事のダッシュボード表示（タイトル、カテゴリ、投稿日、更新日、公開ステータスのリスト化）
* 記事の作成・編集・削除（タイトル、URLスラグ、作成/更新日時、タグ、Markdown本文、下書き/公開ステータス）



**3. 非機能要件・技術スタック**

* **フロントエンド / API**: Next.js (App Router), TypeScript
* **データベース**: Supabase (PostgreSQL)
* **ORM**: Prisma
* **セキュリティ**: Supabase Authによる認証と、Row Level Security (RLS) によるデータアクセス制御
